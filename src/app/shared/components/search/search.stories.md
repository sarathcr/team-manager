# Search

## Description

A convenience directive for applying styling to a search.

## Usage

### Default

```js
<app-search></app-search>
```

### With properties

```js
<app-search
 [placeholder] =" ">
</app-search>
```

## Properties

### Input

| Name              | Type               | Required | Values               |  Default  | Description                                  |
| ----------------- | ------------------ | :------: | -------------------- | :-------: | -------------------------------------------- |
| `placeholder`     | `String`           |    -     | -                    |     -     | To set the placeholder of search component |
